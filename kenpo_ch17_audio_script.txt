# がんばけんぽう だい17しょう「じゅえきけん(こくむせいきゅうけん)」おんせいだいほん
# VOICEVOX しこくめたんよう

===== いんとろ =====

こんにちは。がんばけんぽう、だい17しょうのてーまは「じゅえきけん」です。こくむせいきゅうけんともよばれます。

このしょうでは、せいがんけん・さいばんをうけるけんり・こっかばいしょうせいきゅうけん・けいじほしょうせいきゅうけんの4つをまなびます。

じゅえきけんは、ほかのじんけんとくらべてしゅつだいひんどはひくめです。ただし、さいばんをうけるけんりはけんぽう32じょう、こっかばいしょうせいきゅうけんはけんぽう17じょうで、ぎょうせいほうとのかんれんでしゅつだいされることがあります。

それでは、いっしょにみていきましょう。

===== せくしょん1:せいがんけん(16じょう) =====

まず、けんぽう16じょうのせいがんけんです。

「なんぴとも、そんがいのきゅうさい、こうむいんのひめん、ほうりつ、めいれいまたはきそくのせいてい、はいしまたはかいせいそのほかのじこうにかんし、へいおんにせいがんするけんりをゆうし、なんぴとも、かかるせいがんをしたためにいかなるさべつてきたいぐうもうけない。」

これが16じょうです。

せいがんけんとはなんでしょうか。

せいがんけんとは、なんらかのじこうにつきこっかきかんにようぼうをのべるけんりです。「さんせいけんをほじゅうするきのう」があるともいわれています。

だいじなてんを3つかくにんします。

1つめ、しゅたいのせいげん。

「なんぴとも」とあるとおり、せいがんけんのしゅたいにはせいげんがありません。みせいねんしゃ・がいこくじん・ほうじんも せいがんできます。

2つめ、せいがんないようのせいげん。

ないようにもせいげんがありません。じぶんのりえきにかんけいのないこと、けんぽうのかいはい、てんのうにかんするせいがんもできます。

3つめ、じゅりきかんのぎむ。

せいがんをうけたきかんは、せいがんを「じゅりしてせいじつにしょりするぎむ」をおうにとどまります。

ないようどおりのそちをとるぎむはありません。つまり、ほうてきこうそくりょくはないのです。

よくひっかけにつかわれるぽいんとです。「せいがんをうけたこっかきかんは、そのないようどおりのそちをとるぎむをおう」というせんたくしは、まちがいです。

ちなみに、せいがんけんはめいじけんぽう30じょうでもほしょうされていた、れきしのふるいけんりです。

===== せくしょん2:さいばんをうけるけんり(32じょう) =====

つぎは、けんぽう32じょうのさいばんをうけるけんりです。

「なんぴとも、さいばんしょにおいてさいばんをうけるけんりをうばわれない。」

とてもみじかいじょうぶんですね。

「さいばん」とはなにをさすのでしょうか。

はんれいの たちばでは、32じょうの「さいばん」とは、82じょう(さいばんのこうかい)の「さいばん」とおなじように、「じゅんぜんたるそしょうじけん」をさします。

そしょうじけんとは、じったいてきけんりぎむをしゅうきょくてきにかくていするあらそいのことです。かんたんにいうと、「かちまけ」をきめるあらそいです。

これにたいし、「ひそしょうじけん」は32じょうの「さいばん」にふくまれません。

ひそしょうじけんとは、じったいてきけんりぎむがかくていしたのち、そのぐたいてきないようをさだめていくしほうぎょうせいさーびすのことです。

たとえば、かじしんぱんほう9じょう1こうおつるいのふうふのどうきょぎむにかんするしんぱんは、ひそしょうじけんです。したがって、こうかいほうていにおけるたいしんおよびはんけつというかたちでなくても、82じょう・32じょうにはいはんしません。

つぎに、かんかつちがいのさいばんについてかくにんしましょう。

はんれいは、「32じょうは、そしょうほうでさだめるかんかつけんをゆうするぐたいてきさいばんしょにおいてさいばんをうけるけんりをほしょうしたものではない」としています。

つまり、かんかつちがいのさいばんは「そしょうほういはん(ほうりついはん)」にとどまり、32じょういはん(けんぽういはん)ではありません。

そのほかのじゅうようなてんもかくにんします。

さんしんせいについてです。さいばんはふつうさんしんせいがとられていますが、これはけんぽうにめいぶんでほしょうされているわけではありません。さいばんしょほうでさいようされているにすぎません。

よって、さいばんいんせいどもけんぽう32じょうにはいはんしません。

また、さいばんをうけるけんりは、がいこくじんにもそのせいかくじょうほしょうがおよぶとかいされています。

さいばんをうけるけんりのせいかくについても まとめておきましょう。

けいじじけんでは、さいばんによらなければけいばつをかせられないという「じゆうけんてきなせいかく」をゆうします。

みんじじけん・ぎょうせいじけんでは、こくみんがさいばんしょにうったえをていきすることができ、さいばんしょはこれをきょひできないという「こくむせいきゅうけんてきなせいかく」をゆうします。

===== せくしょん3:こっかばいしょうせいきゅうけん(17じょう) =====

つぎは、けんぽう17じょうのこっかばいしょうせいきゅうけんです。

「なんぴとも、こうむいんのふほうこういにより、そんがいをうけたときは、ほうりつのさだめるところにより、こくまたはこうきょうだんたいに、そのばいしょうをもとめることができる。」

これが17じょうです。

こっかばいしょうせいきゅうけんのいぎについてかくにんしましょう。

めいじけんぽうかでは「こっかむとうせきのげんそく」というものがありました。「こっかがまちがったことをするわけがない。だからせきにんはおわない」というかんがえかたです。したがって、めいじけんぽうにはこっかばいしょうせいきゅうけんのきていはありませんでした。

にほんこくけんぽう17じょうは、このこっかむとうせきのげんそくへのはんせいからきていされました。

こっかばいしょうせいきゅうけんのこうぞうについてみてみましょう。

まず、しゅたいです。「なんぴとも」とあり、がいこくじんにもほしょうされています。

つぎに、あいてかたです。「こくまたはこうきょうだんたいに」とあり、こうむいんこじんへのちょくせつせいきゅうはできません。

こくまたはこうきょうだんたいのせきにんは「だいいせきにん」です。こうむいんがしょくむをおこなうについて、こいまたはかしつによりそんがいをあたえたときに、こくまたはこうきょうだんたいがかわりにせきにんをおうということです。

「ばいしょう」と「ほしょう」のちがいもかくにんしましょう。

「ばいしょう」は、こくまたはこうきょうだんたいのこういが「いほう」なばあいのそんがいのてんぽです。

「ほしょう」は、こくまたはこうきょうだんたいのこういが「てきほう」なばあいのそんしつのてんぽです。けんぽう29じょう3こうの「せいとうなほしょう」がこれにあたります。

ぐたいかりっぽうとして、こっかばいしょうほうがあります。これはぎょうせいほうのちょうひんしゅつぶんやです。

さいごに、じゅうようなはんれいをみてみましょう。

ゆうびんほういけんはんけつ(さいだいはんへいせい14ねん9がつ11にち)です。

このはんけつでは、2つのはんだんがなされました。

1つめ、しょりゅうゆうびんぶつについて。ゆうびんぎょうむじゅうじしゃの「こいまたはじゅうだいなかしつ」によってそんがいがじっしたばあいに、こっかのそんがいばいしょうせきにんをめんじょ・せいげんしているぶぶんは、けんぽう17じょうにいはんする。

2つめ、とくべつそうたつゆうびんぶつについて。ゆうびんぎょうむじゅうじしゃの「けいかしつ」によるふほうこういにもとづきそんがいがじっしたばあいにも、こっかのそんがいばいしょうせきにんをめんじょ・せいげんしているぶぶんは、けんぽう17じょうにいはんする。

ゆうびんほういけんはんけつはたいへんじゅうようなはんれいです。しっかりおさえておいてください。

===== せくしょん4:けいじほしょうせいきゅうけん(40じょう) =====

さいごは、けんぽう40じょうのけいじほしょうせいきゅうけんです。

「なんぴとも、こうりゅうまたはこうきんされたのち、むざいのさいばんをうけたときは、ほうりつのさだめるところにより、こくにそのほしょうをもとめることができる。」

これが40じょうです。

けいじほしょうせいきゅうけんのようけんを2つかくにんします。

1つめ、「こうりゅうまたはこうきんされたのち」、つまりみがらこうそくがあること。

2つめ、「むざいのさいばんをうけた」こと。

こうしきであらわすと、「みがらこうそく＋むざいはんけつ＝けいじほしょうせいきゅうか」となります。

じゅうようなはんれいをみてみましょう。

まず、「こうりゅうまたはこうきん」のはんいについてです。

ふきそとなったじじつにもとづくこうりゅう・こうきんであっても、そのなかに、じつしつじょうむざいとなったじじつについてのこうりゅう・こうきんとみとめられるものがふくまれるときは、そのぶぶんも40じょうの「こうりゅうまたはこうきん」にあたります(さいだいけつしょうわ31ねん12がつ24にち)。

つぎに、「むざいのさいばん」の はんいについてです。

しょうねんしんぱんのふしょぶんけっていは、40じょうの「むざいのさいばん」にあたらず、けいじほしょうのたいしょうにはなりません(さいけつへいせい3ねん3がつ29にち)。

===== はんれいまとめ =====

だい17しょうにかんするはんれいをまとめます。

1つめ、ゆうびんほういけんはんけつ(さいだいはんへいせい14ねん)。
ろんてんは、ゆうびんほうによるこっかのばいしょうせきにんのめんじょ・せいげん。
けつろんは、しょりゅうゆうびんについてこい・じゅうだいかしつのばあい、とくべつそうたつゆうびんについてけいかしつのばあいの めんじょ・せいげんは、17じょうにいはんする。

2つめ、かんかつちがいのさいばんにかんするはんれい(さいだいはんしょうわ24ねん)。
ろんてんは、32じょうとかんかつちがい。
けつろんは、かんかつちがいのさいばんはそしょうほういはんにとどまり、32じょういはんではない。

3つめ、こうりゅう・こうきんのはんいにかんするはんれい(さいだいけつしょうわ31ねん)。
ろんてんは、40じょうの「こうりゅうまたはこうきん」のはんい。
けつろんは、ふきそじじつにもとづくものでも、じつしつじょうむざいぶぶんについてのものはふくまれる。

===== まとめ =====

それでは、だい17しょうのまとめです。

1. せいがんけん(16じょう)のしゅたいにせいげんはなく、じゅりきかんはせいじつにしょりするぎむをおうにとどまる。ないようどおりのそちぎむはない。

2. さいばんをうけるけんり(32じょう)の「さいばん」はじゅんぜんたるそしょうじけんをさし、ひそしょうじけんはふくまれない。かんかつちがいのさいばんはそしょうほういはんにとどまり、32じょういはんではない。

3. こっかばいしょうせいきゅうけん(17じょう)は、めいじけんぽうのこっかむとうせきのげんそくへのはんせいからきていされた。せいきゅうあいてはこくまたはこうきょうだんたいで、こうむいんこじんへのちょくせつせいきゅうはできない。

4. けいじほしょうせいきゅうけん(40じょう)のようけんは、みがらこうそく＋むざいはんけつのふたつ。しょうねんしんぱんのふしょぶんけっていはむざいのさいばんにあたらない。

じゅえきけんは、こうみんのけんりをまもるためのたいせつなせいどです。

とくに、こっかのちからによってそんがいをうけたばあいに、こっかにたいしてきゅうさいをもとめることができるというてんで、みんしゅしゅぎのきばんをなすけんりです。

だい17しょう、じゅえきけん、これでおわりです。

おつかれさまでした!

えんしゅうもんだいでちしきをかくにんしてくださいね。
